ЗАДАЧА 2: Одна лента действий + быстрый «+ Добавить» (главный экран Ани)

КОНТЕКСТ
Сейчас «что делать» дублируется в 4 местах: «Сейчас важно» на Главной, раздел
Задачи, раздел Напоминания и «Таблетки на сегодня». Плюс ввод данных раскидан по
разделам. Сводим в одну систему. Это сторона АНИ; Главную Дена не трогаем (у него
уже читательский бриф), приватность/роли тоже не трогаем (сделано в задачах 1–1.3).

ЧТО СДЕЛАТЬ
1. «Сейчас важно» на Главной Ани = ЕДИНАЯ лента действий на сегодня (очередь).
   В неё сводятся: приём таблеток на сегодня, визиты/врачи на сегодня, напоминания
   на сегодня, дни рождения, задачи (из бывшего раздела «Задачи» и из встреч
   «Дела→напоминания»). Пункт отмечается выполненным прямо из ленты → гаснет/уходит.
   Отметка меняет состояние источника (отметил таблетку в ленте → отмечено и в
   «Здоровье»). Никаких двух чеклистов на сегодня.
2. Раздел «Задачи» как отдельный пункт — убрать. Всё живёт в ленте.
3. «Напоминания» — оставить как ДВИЖОК/НАСТРОЙКИ правил (разовые + повторяющиеся),
   который ПИТАЕТ ленту. Это место, где задаются правила (вода 08/12/18 и т.п.),
   а не второй список дел.
4. Таблетки — детальный трекер остаётся в «Здоровье»; на сегодня они появляются в
   ленте один раз.
5. Карточки-рекомендации: под лентой показывать карточки с ПРИЧИНОЙ и ДЕЙСТВИЕМ
   (брать из данных брифа, которые уже считаются), например:
   • ⚠️ «Сон падает 3 дня → предлагаю перенести тяжёлые встречи»
   • 🟢 «Сегодня хороший день для переговоров»
   • 🎁 «ДР через 2 дня → поздравление готово»
   Карточка раскрывается в детали. Это «рекомендация + почему + что сделать»,
   а не сырой список.
6. Единый «+ Добавить»: заметная плавающая кнопка (FAB) на Главной Ани. Нажатие →
   меню быстрого ввода: Встреча · Контакт · Расход · Состояние · Анализ · Лекарство
   · Задача · Заметка. Каждый пункт ведёт на СУЩЕСТВУЮЩУЮ форму добавления (где
   формы нет — простая форма). Это главный быстрый путь ввода для оператора.

КРИТЕРИИ ГОТОВНОСТИ
- На Главной Ани одна лента «Сейчас важно» из всех источников, без повторов.
- Отдельного раздела «Задачи» в меню больше нет.
- «Напоминания» = настройка правил, питает ленту.
- Приём таблетки виден в ленте один раз; детальный трекер — в «Здоровье»;
  отметка синхронна.
- Есть карточки-рекомендации с причиной и действием.
- Есть плавающая кнопка «+ Добавить» с быстрым меню ввода.
- У Дена лента по-прежнему без медицинских пунктов (как в задаче 1).

НЕ ДЕЛАТЬ: не трогать роли/приватность; не возвращать фокус/настроение;
Ден остаётся на просмотре.
